<h1 align="center">Workspaces</h1>

<p align="center">Switch between your project folders without leaving VS Code.</p>

## Features

- **Sidebar panel** -- browse your saved workspaces in the activity bar
- **Quick picker** -- press `Cmd+Alt+W` (`Ctrl+Alt+W` on Windows/Linux) to jump to any workspace
- **Drag and drop** -- reorder workspaces however you like
- **Zero config** -- no files to manage, everything is stored in VS Code's global state

## Usage

1. Open the **Workspaces** panel in the sidebar
2. Click **+** to add folders
3. Click a workspace to open it
